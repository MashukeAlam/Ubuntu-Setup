# A script for setting up Ubuntu for development according to MY need.

Feel free to change the script up here and there to fit to your need. This script will install the following software and is tested to do so properly on Ubuntu 20.04 system. 

**Credit goes to this repo [(Ubuntu-Dev-Setup)](https://github.com/lukemorales/ubuntu-dev-setup) by [Like Morales](https://github.com/lukemorales)**

**For some reason Ruby on Rails installation is not working**




# Things that this script will do

 1. Set up git username and email
 2. Update the repository
 3. Install Curl
 4. Install neofetch
 5. Install copyq
 6. Install git
 7. Install python3 pip
 8. Install getgist
 9. Install Firacode
 10. Install macOS themes & icons & wallpapers
 11. Install Gnome Tweaks Tool
 12. Install NVM
 13. Install NodeJS latest
 14. Install yarn
 15. Install RBENV
 16. Install Ruby & Bundler & Rails (**Currently not working**)
 17. Install VS Code
 18. Install Insomnia Core
 19. Install MPV
 20. Install OBS
 21. Install KVM
 22. Install DBeaver
 23. Install fzf
 24. Install Peek 
 25. Install Google Chrome
 26. Add some aliases
 27. Clean unnecessary packages 
 28. Download SF Pro Fonts 
 29. Download Monaco Fonst
